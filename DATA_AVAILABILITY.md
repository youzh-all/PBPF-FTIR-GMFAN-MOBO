# Data availability

This repository provides manuscript-aligned metadata, minimal representative sample data, and a lightweight public consistency-check script for the PBPF FT-IR/GMFAN/MOBO workflow.

The representative sample data are provided to demonstrate data format and repository workflow only. They are not intended to reproduce the full model performance reported in the manuscript.

The full FT-IR, DSC, and UTM datasets used for model training and evaluation are available from the corresponding authors upon reasonable request.
