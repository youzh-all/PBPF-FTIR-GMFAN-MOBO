# Representative sample data

This folder contains minimal representative sample files for public file-format demonstration only.

The sample FT-IR file contains one downsampled multiplicative-scatter-corrected spectrum for selected PBPF formulation states. It is intentionally small and is not intended to reproduce the full model performance reported in the manuscript.

The full FT-IR, DSC, and UTM datasets used for training and evaluation are not included in this public package and are available from the corresponding authors upon reasonable request.
