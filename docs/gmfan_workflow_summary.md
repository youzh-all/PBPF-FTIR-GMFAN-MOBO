# GMFAN workflow summary

The manuscript describes a gated multimodal fusion attention network (GMFAN) that combines FT-IR spectral descriptors with encoded formulation descriptors to infer DSC heating, DSC cooling, and UTM stress-strain response curves.

This public repository does not expose the full internal training pipeline or checkpoints. Instead, it preserves manuscript-aligned metadata and representative input-format samples so that readers can inspect the formulation space, split policy, uncertainty summary, and computationally prioritized MOBO candidates.
